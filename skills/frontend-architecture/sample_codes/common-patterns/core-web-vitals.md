# Core Web Vitals Examples

Examples and patterns to improve LCP, FID/INP, and CLS. Move common techniques into components or build-time optimizations.

- preload critical assets, use `fetchpriority` / `loading="lazy"`
- break up long tasks and yield to the browser
- reserve space for dynamic content with dimensions or `aspect-ratio`

See `image-optimization.md` and `lazy-loading.jsx` for runnable snippets.
